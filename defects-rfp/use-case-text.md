# Что покрывают акторы и кейсы

- Диспетчер: смотрит свободные места и доступных работников, направляет авто в зону, ищет кейсы.
- Бригадир: назначает ремонт конкретному работнику, получает сменный отчёт по бригаде.
- Ремонтник: отмечает доступность на смену, проводит диагностику, старт/финиш ремонта, прикрепляет фото.
- ОТК (QC): регистрирует дефекты (после испытаний), фиксирует результаты тестов, получает сменный отчёт.
- Начальник цеха: получает сменный агрегированный отчёт.
- Бухгалтерия: месячный отчёт по отработанным сменам.
- Администратор: ведёт справочники зон/мест/бригад/бригадиров.
- ERP/MES (внешняя): поставляет контекст заказа/плана при регистрации дефекта.
