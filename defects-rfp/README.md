# Диаграммы Mermaid

- диаграммы .md созданы на онлайн платформе https://www.mermaidchart.com/
  и выгружены в код
- чтобы просмотреть не код, а диаграммы в VScode нужно установить расширение `Markdown Preview Mermaid Support`
- VSCode покажет визуализацию прямо в предпросмотре `Markdown Ctrl+Shift+V (или Cmd+Shift+V на Mac)`

# Диаграммы Plant

- файлы .wsd созданы на платформе https://editor.plantuml.com/uml и выгружены в код
- чтобы просмотреть не код, а диаграммы в VScode нужно установить расширение `PlantUML`
- на текущем файле сделать cmd+P и ввести `> PlantUML: Preview Current diagram`
